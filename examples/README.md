# Examples

Runnable reference components, type-correct against the public `@plurid/plurid-react` API. They are
copy-pasteable single files (no build wiring) — drop one over a `fixtures/render-test`-style host, or into
your own React 19 app, and render it.

| Example | What it shows |
|---|---|
| [`minimal`](./minimal) | Hello-world: three planes in a 3D space, zero configuration. Start here. |
| [`control-surface`](./control-surface) | A small "spatial notes" component exercising **every** control seam — `onReady`, pub/sub control & observe, a storage adapter, tuned timings, gesture `buttonMap`, shortcuts, and a custom toolbar slot. |

New to the engine? Read [`../GETTING_STARTED.md`](../GETTING_STARTED.md) first, then come back for the
full reference in [`../docs/CONTROL_SURFACE.md`](../docs/CONTROL_SURFACE.md).
